WHAT YOU GOT:
-------------

A web-based ham radio operations portal for Macomb County, Michigan.

Features:
✓ Multi-room live chat with other users
✓ Real-time weather radar (live precipitation data)
✓ Emergency alert system (for Owner/Moderators)
✓ Meeting scheduler and event calendar
✓ Operator check-in and status tracking
✓ County repeater frequencies and tones
✓ Login activity logging (for admins)
✓ Dark/Light theme toggle
✓ Automatic session persistence (stay logged in)
✓ 3-attempt login security with lockout protection
✓ Works offline (except weather radar)

QUICK START:
------------

1. Double-click ARPSC-START.bat
2. Browser opens automatically
3. Enter your full name (required)
4. Enter PIN from your administrator
5. Enter callsign (optional)
6. Click Login
7. App is ready to use

STAY LOGGED IN:
---------------

The app remembers you automatically.
Close the app, restart browser, restart computer... you stay logged in.
You only need to log in again after clicking "LOGOUT" button.

NEED HELP?
----------

1. Read USER-GUIDE.txt (in this folder)
2. Contact your ARPSC administrator
3. Try the other browser (Chrome vs Edge)
