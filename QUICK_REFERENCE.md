## 📱 6 App Tabs

1. **🏠 Home** - Welcome screen, user info, quick stats
2. **💬 Chat** - Multi-room chat, reactions, messaging
3. **👥 Members** - Search members, view callsigns
4. **📅 Events** - Club events and activities
5. **ℹ️ Info** - Club information, frequencies
6. **⚙️ Settings** - Status, dark mode, notifications

---

## 💬 Chat Features

### Switch Rooms:
- Tap room header → Select room

### Send Message:
- Type → Tap send button

### Add Reaction:
- Long-press message → Tap emoji

### Delete Message:
- Tap trash icon (own messages)
- Tap delete (Mod/Owner: any message)

### Create Room (Mod/Owner):
- Tap room header → "Create New Room"

---

## 🎨 Themes

### Toggle Dark Mode:
Settings → Appearance → Dark Mode switch

### Features:
- Light mode: Blue/white
- Dark mode: Blue/black
- All screens themed
- Instant switching
